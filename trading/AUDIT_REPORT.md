### 2.3 SQL 注入详情 (S-01)

```python
# 当前代码 (database.py:92)
f"INSERT INTO trade_records({columns}) VALUES({placeholders})"

# 风险: columns 来自 kwargs 的 key，如果外部传入恶意列名可注入
# 修复: 添加列名白名单
ALLOWED_COLUMNS = {"order_id", "symbol", "side", "direction", "price", ...}
invalid = set(kwargs.keys()) - ALLOWED_COLUMNS
if invalid:
    raise ValueError(f"非法列名: {invalid}")
```
